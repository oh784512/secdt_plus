# SecDT+ PoC prototype
* This is the prototype for SecDT+

## Set Up
```shell=
pip install -r requirements.txt
```

## Run the Demo
Run the demo python script.
```shell=
python secdtplus/secdt_demo.py
```

Choose testing data set. (1~6, default=3) 
```shell=
python secdtplus/secdt_demo.py -s 3
```
